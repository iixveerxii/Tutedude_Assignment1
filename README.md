# Tutedude_Assignment1
Submission of First Assignment of tutedude python program
Assignment1_Task1.py Takes two numbers as input from the user. Performs Addition, Subtraction, Multiplication, Division. Displays the results of each operation on the screen.
Assignment1_Task2.py Takes a user's first name and last name as input. Concatenates into a full name. Displays a personalized greeting message using the full name.
